# 🧶 Pontinhos da Manu

**Pontinhos da Manu** é uma loja online simples e elegante para venda de peças artesanais de crochê feitas à mão.

---

# ✨ Demonstração

O site possui:

* Página de produtos
* Cards de produtos com preço
* Avaliações de clientes
* Cálculo de frete por CEP
* Botão para Instagram
* Seção de contato
* Animação de transição ao entrar na página

---

# 🖼️ Preview

Interface focada em estilo artesanal:

* cores suaves
* cards modernos
* navegação simples
* layout responsivo

---

# 🚀 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* API pública **ViaCEP** para consulta de CEP

---

# 📦 Funcionalidades

### 🛍️ Produtos

* Grid responsivo de produtos
* Imagem, descrição e preço
* Botão de compra

### 🚚 Cálculo de frete

* Campo para digitar CEP
* Consulta automática de cidade e estado
* Estimativa de frete e prazo

### ⭐ Avaliações

* Cards de clientes
* Estrelas de avaliação
* Layout responsivo

### 📱 Redes sociais

* Botão direto para Instagram

### 📬 Contato

* Links para contato direto com a loja
